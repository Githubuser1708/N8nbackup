# N8N Backup
This repository contains workflows created in n8n and yhey are automatically backed up on schedule.
Some of the Workflows:

1. WhatsappChat
   - Workflow to receive whatsapp messages, processed by AI AGent and replies via whatsapp messages.
  
2. Ai Agent Task Automation
   - Workflow to receive messages via chat and OpenAI Chat will process the messages and reply accordingly.
